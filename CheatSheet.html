3. Identify attributes

HTML attributes provide additional information about HTML elements.

All HTML elements can have attributes

Attributes provide additional information about elements

Attributes are always specified in the start tag Attributes usually come in name/value pairs like: name="value"

Examples are the:

href Attribute

The a tag defines hyperlink. The href attribute specifies the URL of the page.
written: (lesser sign) a href="example.com" (lesser sign) /a (greater sign)

src Attribute

The img tag is used to embed an image in an HTML page. The src attribute specifies the path to the image.
written: (lesser sign) img src="example.jpg" (greater sign)

width and height Attribute

The img tag should also contain the width and height attributes, which specifies the width and height of the image (in pixels).
written: (lesser sign) img src="example.jpg" width="(width in pixel)" height="(height in pixel)" (greater sign)

alt Attribute

The required alt attribute for the img tag specifies an alternate text for an image, if the image for some reason cannot be displayed.

This can be due to slow connection, or an error in the src attribute, or if the user uses a screen reader.
written: (lesser sign) img src="example.jpg" alt="(text used instead of the image)" (greater sign)

style Attribute

The style attribute is used to add styles to an element, such as color, font, size, and more.
written: (lesser sign) p style="color:red;" (greater sign)

lang Attribute

You should always include the lang attribute inside the html tag, to declare the language of the Web page.

This is meant to assist search engines and browsers.
written: (in first html tag) (lesser sign) html lang="en" (greater sign) for US english use: en-US

title Attribute

The title attribute defines some extra information about an element.

The value of the title attribute will be displayed as a tooltip when you mouse over the element.
written: (used in the header or paragraph) (lesser sign) p title="Text when you mouse over it" (greater sign) text (lesser sign)(slash)p(greater sign)
